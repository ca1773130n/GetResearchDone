  <action>
    **Step 1: Add cmd* CLI entry point functions to lib/evolve.js**

    Add a new section `// ─── CLI Command Functions ─────────────────────────────────────────────────`

    1. `cmdEvolveDiscover(cwd, args, raw)` — runs discovery and outputs results:
       - Parse `--count N` flag from args (default: DEFAULT_ITEMS_PER_ITERATION)
       - Call `readEvolveState(cwd)` to get previous state (may be null)
       - Call `runDiscovery(cwd, previousState)`
       - Output JSON: `{ selected, remaining, all_discovered_count, merged_count, items_per_iteration }`
       - Raw output: `selected.length` items as count string

    2. `cmdEvolveState(cwd, _args, raw)` — reads and outputs the current evolve state:
       - Call `readEvolveState(cwd)`
       - If null, output `{ exists: false, state: null }`
       - If exists, output `{ exists: true, state: <full state object> }`
       - Raw output: JSON.stringify of state or 'No evolve state found'

    3. `cmdEvolveAdvance(cwd, _args, raw)` — advances to next iteration:
       - Call `readEvolveState(cwd)`
       - If null, output error: 'No evolve state found. Run discover first.'
       - Call `advanceIteration(previousState)`
       - Call `writeEvolveState(cwd, newState)`
       - Output the new state JSON
       - Raw output: `iteration ${newState.iteration}` string

    4. `cmdEvolveReset(cwd, _args, raw)` — deletes the evolve state file (start fresh):
       - Get path via `evolveStatePath(cwd)`
       - If file exists, delete it with `fs.unlinkSync`
       - Output `{ reset: true }`
       - Raw output: 'Evolve state reset'

    Add all 4 cmd* functions to `module.exports`.

    **Step 2: Add cmdInitEvolve to lib/context.js**

    Add import at the top of context.js:


    Add a new function `cmdInitEvolve(cwd, raw)`:
    ```js
    /**
     * CLI command: Initialize context for the evolve workflow.
     * @param {string} cwd - Project working directory
     * @param {boolean} raw - Raw output mode
     */
    function cmdInitEvolve(cwd, raw) {
      const config = loadConfig(cwd);
      const backend = detectBackend(cwd);
      const capabilities = getBackendCapabilities(backend);

      // Read existing evolve state (may be null if first iteration)
      const evolveState = readEvolveState(cwd);

      // Resolve models — evolve uses sonnet-tier ceiling (enforced in Phase 56)
      const plannerModel = resolveModelForAgent(config, backend, 'grd-planner');
      const executorModel = resolveModelForAgent(config, backend, 'grd-executor');

      const result = {
        backend,
        capabilities,
        config: {
          model_profile: config.model_profile || 'balanced',
          autonomous_mode: config.autonomous_mode || false,
          items_per_iteration: (evolveState && evolveState.items_per_iteration) || 5,
        },
        evolve_state: {
          exists: !!evolveState,
          iteration: evolveState ? evolveState.iteration : 0,
          remaining_count: evolveState ? (evolveState.remaining || []).length : 0,
          bugfix_count: evolveState ? (evolveState.bugfix || []).length : 0,
        },
        models: {
          planner: plannerModel,
          executor: executorModel,
        },
        milestone: getMilestoneInfo(cwd),
      };

      output(result, raw, raw ? JSON.stringify(result) : undefined);
    }
    ```

    Add `cmdInitEvolve` to the `module.exports` at the bottom of context.js.
