    Add cmdMigrateDirs function to lib/commands.js. Import currentMilestone and milestonesDir from ./paths (already partially imported — add currentMilestone to the existing require('./paths') destructuring).

    Function signature: cmdMigrateDirs(cwd, raw)

    Implementation logic:

    1. Detect current milestone via currentMilestone(cwd). This returns e.g. 'v0.2.1' or 'anonymous'.

    2. Define the migration map (5 directories):
       - .planning/phases/ -> .planning/milestones/{milestone}/phases/
       - .planning/research/ -> .planning/milestones/{milestone}/research/
       - .planning/codebase/ -> .planning/milestones/{milestone}/codebase/
       - .planning/todos/ -> .planning/milestones/{milestone}/todos/
       - .planning/quick/ -> .planning/milestones/anonymous/quick/ (always anonymous, not current milestone)

    3. For each entry in the map:
       a. Check if old-style directory exists (fs.existsSync)
       b. If not, skip (add to skipped array)
       c. Check if old-style directory has any contents (fs.readdirSync)
       d. If empty, skip
       e. Ensure target parent directory exists (fs.mkdirSync recursive)
       f. Move contents: for each entry in old directory, use fs.cpSync (recursive) to copy to target, then fs.rmSync the old entry. Do NOT remove the old directory itself (leave it empty for backward compat during transition).
       g. If target directory already has files, MERGE — only copy entries that don't already exist at the target (prevents overwriting).
       h. Add to moved_directories array

    4. Idempotency check: If none of the 5 old-style directories have content (all empty or missing), set already_migrated to true.

    5. Output result JSON:
       {
         milestone: detected milestone,
         moved_directories: [ { from: "phases", to: "milestones/v1.0/phases", entries_moved: N } ],
         skipped: [ "codebase" ],
         already_migrated: boolean,
         errors: []
       }

    Add cmdMigrateDirs to module.exports.

    Run tests — they MUST PASS (GREEN phase).
