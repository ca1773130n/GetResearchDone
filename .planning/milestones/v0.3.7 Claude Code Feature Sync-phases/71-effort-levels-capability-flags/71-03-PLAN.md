    Update the return object in `buildInitContext`:

    ```typescript
    function buildInitContext(
      cwd: string,
      overrides: Record<string, unknown>
    ): Record<string, unknown> {
      const backend = detectBackend(cwd);
      const milestone = getMilestoneInfo(cwd);
      const phases_dir = path.relative(cwd, getPhasesDirPath(cwd));
      const research_dir = path.relative(cwd, getResearchDirPath(cwd));
      const caps = getBackendCapabilities(backend);
      return {
        backend,
        backend_capabilities: caps,
        effort_supported: caps.effort === true,
        milestone_version: milestone.version,
        milestone_name: milestone.name,
        phases_dir,
        research_dir,
        ...overrides,
      };
    }
    ```

    Note: `caps` is extracted into a variable so it can be used for both `backend_capabilities` and the `effort_supported` shortcut. The `=== true` guard handles the case where `effort` might be a string or undefined in edge cases.
