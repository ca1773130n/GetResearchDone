       CRITICAL: All cmd* functions call `output()` which calls `process.exit(0)`. Unit tests need:

       - Export both as reusable helpers
       - Implementation: jest.spyOn(process, 'exit').mockImplementation((code) => { throw { __EXIT__: true, code }; })
       - jest.spyOn(process.stdout, 'write').mockImplementation((data) => { captured += data; })
       - Wrap fn call in try/catch, check for __EXIT__ sentinel

       - `createFixtureDir()` — Copies tests/fixtures/planning/ into a temp dir under os.tmpdir(), returns the temp root path

       - Use fs.mkdtempSync and fs.cpSync (Node 16.7+) or recursive manual copy
       - The fixture has .planning/ subdirectory with config.json, STATE.md, ROADMAP.md, phases, todos
