    8. If result has `error` property: `output({ error: result.error }, raw, result.error)`.
    9. Otherwise: output `{ content: result, path: '.planning/LONG-TERM-ROADMAP.md', version, updated_fields: Object.keys(updates) }`.
    10. Raw mode: output the updated markdown content.

    **Subcommand: `promote`**
    ```
    node grd-tools.js long-term-roadmap promote --version <version>
    ```
    1. Parse `--version` flag.
    2. If no version: `error('--version flag required')`.
    3. Read `.planning/LONG-TERM-ROADMAP.md`.
    4. If not found: output error.
    5. Call `promoteMilestone(content, version)`.
    6. If result has `error` property: `output({ error: result.error }, raw, result.error)`.
    7. Otherwise: Call `getMilestoneTier(result, version)` to get the new tier.
       Output `{ content: result, path: '.planning/LONG-TERM-ROADMAP.md', version, new_tier: newTier }`.

    ```
    node grd-tools.js long-term-roadmap history --action <action> --details <details>
    ```

    1. Parse `--action` and `--details` flags.
    2. If no action: `error('--action flag required')`.
