    Create `lib/paths.js` following the established module pattern (see lib/utils.js, lib/state.js for reference):

    **Module header:**
    ```javascript
    /**
     * GRD Path Resolution — Centralized .planning/ subdirectory path construction
     *
     * Single source of truth for all milestone-scoped directory paths.
     * All other modules should import from this module instead of
     * hardcoding path.join(cwd, '.planning', 'phases', ...) constructions.
     *
     * Dependencies: fs (Node built-in only — no circular lib/ deps)
     */
    'use strict';

    const fs = require('fs');
    const path = require('path');
    ```

    **IMPORTANT: No lib/ dependencies.** This module must NOT `require('./utils')` or `require('./state')` to avoid circular dependency issues. It reads STATE.md directly using `fs.readFileSync`. This is intentional — paths.js is the lowest-level module that everything else depends on.

    **currentMilestone(cwd):**
    - Read `path.join(cwd, '.planning', 'STATE.md')` using `fs.readFileSync`
    - Use regex `\*\*Milestone:\*\*\s*(.+)` to extract the Milestone field value
    - From the field value (e.g., `v0.2.1 — Hierarchical Planning Directory`), extract just the version string: match `(v[\d.]+)` from the field value
    - If STATE.md is missing, unreadable, has no Milestone field, or field has no version string: return `'anonymous'`
    - This function does NOT use `stateExtractField` from state.js — it reads STATE.md directly to avoid circular deps

    **milestonesDir(cwd):**
    - Returns `path.join(cwd, '.planning', 'milestones')`
    - Pure computation, no filesystem access

    **phasesDir(cwd, milestone):**
    - If `milestone` is `undefined` or `null`: set `milestone = currentMilestone(cwd)`
    - Returns `path.join(cwd, '.planning', 'milestones', milestone, 'phases')`

    **phaseDir(cwd, milestone, phaseDirName):**
    - If `milestone` is `undefined` or `null`: set `milestone = currentMilestone(cwd)`
    - Returns `path.join(cwd, '.planning', 'milestones', milestone, 'phases', phaseDirName)`

    **researchDir(cwd, milestone):**
    - If `milestone` is `undefined` or `null`: set `milestone = currentMilestone(cwd)`
    - Returns `path.join(cwd, '.planning', 'milestones', milestone, 'research')`

    **codebaseDir(cwd, milestone):**
    - If `milestone` is `undefined` or `null`: set `milestone = currentMilestone(cwd)`
    - Returns `path.join(cwd, '.planning', 'milestones', milestone, 'codebase')`

    **todosDir(cwd, milestone):**
    - If `milestone` is `undefined` or `null`: set `milestone = currentMilestone(cwd)`
    - Returns `path.join(cwd, '.planning', 'milestones', milestone, 'todos')`

    **quickDir(cwd):**
    - Always returns `path.join(cwd, '.planning', 'milestones', 'anonymous', 'quick')`
    - Does NOT accept a milestone parameter — quick tasks are always under anonymous

    **archivedPhasesDir(cwd, version):**
    - Returns `path.join(cwd, '.planning', 'milestones', version + '-phases')`
    - This matches the existing archive layout (e.g., `.planning/milestones/v0.1.6-phases/`)
