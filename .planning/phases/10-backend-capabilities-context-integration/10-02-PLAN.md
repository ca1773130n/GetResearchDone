    **Step 2: Add `backend` and `capabilities` to each cmdInit* function's result object**


    ```js

    backend_capabilities: getBackendCapabilities(backend),
    ```

    7. `cmdInitVerifyWork` — same pattern
    8. `cmdInitPhaseOp` — same pattern
    9. `cmdInitTodos` — same pattern
    10. `cmdInitMilestoneOp` — same pattern
    11. `cmdInitMapCodebase` — same pattern
    12. `cmdInitProgress` — same pattern
    13. `cmdInitResearchWorkflow` — same pattern
    14. `cmdInitPlanMilestoneGaps` — same pattern
